# Mobile-Layout
移动开发必备布局框架

## Files
此项目只有两个主要文件

| 文件 | 介绍
| --- | --- | ---
| ui-base.css | 基础文件，必须引入的文件
| ui-box.css | 布局框架

## ui-base.css
文档地址 : http://newdocx.appcan.cn/UI/base
## ui-box.css
使用弹性盒子对页面进行布局，ui-box.css中就定义了弹性盒子的一些css3样式，弹性盒子模型是在指定大小的父容器里来为子元素分配空间，使用box架构可以更容易更方便的适配不同分辨率不同屏幕尺寸的手机
文档地址：http://newdocx.appcan.cn/UI/box
